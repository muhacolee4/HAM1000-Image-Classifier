# HAM1000-Image-Classifier
These program will sort images in the HAM1000 date set into different folders according to their identifier.

Steps on how to use the program
--------------------------------
1. Open the installed program (HAM10000) ie. click on HAM10000 Image Classifier on desktop
2. Unzip the downloaded HAM10000 images
3. Join all images into one folder i.e all images/
4. Enter the source address i.e users/all_images/
5. Enter destination address where you want the images to be arranged i.e /reorganized if my destination folder is reorganized folder
6. Click classify.
